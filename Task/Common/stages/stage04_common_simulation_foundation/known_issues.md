# Common-04 Known Issues

- No BCH, convolutional-code or LDPC implementation is included in Common-04.
- No interleaving or deinterleaving implementation is included in Common-04.
- Formal capacity is validated as configuration support for 50000 frames, 1000 symbols/frame and 50 shards. Full formal curves are not run in this Stage.
- Large binary noise pools and generated PNG files are not committed.
- Gaussian sanity checks are wide finite-sample checks and are not a proof of normality.
